# Push_swap

## Description

The **push_swap** assignment is about making a function that prints commands that sorts a list of numbers in ascending order by using two stacks. The main goal of this assignment is learning different types of sorting algorithms as well as implementing what you have learned so far.

You are given 11 commands to sort stack_a. `s` for "swap", `r` for "rotate", `p` for push, followed by `a` or `b` to imply which stack to operate on. `ss`, `rr`, `rrr` mean both stacks. Notice how `p` does not have an option for both stacks. `rr` folowed by a third letter is for reverse rotation.
You sort stack_a by printing these commands. If the amount of commands are less than a set limit, you have completed the assignment, given stack_a is sorted correctly with no leaks from allocation.

## Instructions

1. Run `make` in the root directory to get the executable `./push_swap`
2. Download the checker program called **checker_Linux** for testing
3. Happy testing!
